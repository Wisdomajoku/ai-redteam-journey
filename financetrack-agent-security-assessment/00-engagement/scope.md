# Scope of Engagement

## Engagement Information

**Client:** FinanceTrack Ltd. (Fictional)

**Assessment:** AI Agent Security Assessment

**Assessment Period:** July 2026

**Assessor:** Wisdom Ajoku

## Objective

FinanceTrack Ltd. requested an independent security assessment of its AI-powered transaction assistant to identify weaknesses that could affect the confidentiality, integrity, or availability of customer information. The assessment evaluates the behaviour of the AI agent under adversarial input and reviews the security controls surrounding its interaction with backend tools.

## In Scope

The assessment includes:

* FinanceTrack AI Transaction Assistant
* Streamlit web application
* LangChain ReAct agent
* Agent tool invocation workflow
* SQLite transaction data accessed through agent tools
* LiteLLM integration
* Prompt handling and agent decision-making behaviour

## Out of Scope

The following components and activities are excluded from this assessment:

* Groq infrastructure and managed services
* Host operating system security
* Network infrastructure
* Denial of service testing
* Source code review
* Third-party services outside the assessment environment

## Assessment Method

Testing is performed using a combination of manual application testing and automated AI red teaming techniques. The assessment includes reconnaissance, automated adversarial testing, targeted multi-turn attacks, manual verification of identified issues, and reporting.

## Deliverables

The engagement produces the following deliverables:

* Engagement documentation
* Reconnaissance documentation
* Automated assessment results
* Manual verification evidence
* Executive summary
* Technical assessment report
* Remediation recommendations
